Samurai Skills Overview

Global Cooldown Actions (Single Target)

Icon Name Description
Hakaze Single target combo starter. 150 potency. +5 Kenki.
Jinpu Single target second step combo. 100 potency, 250 potency if combo from Hakaze. Combo Bonus: Fugetsu buff (increased damage dealt by 13% for 40s), +5 Kenki.
Shifu Single target second step combo. 100 potency, 250 potency if combo from Hakaze. Combo Bonus: Fuka buff (reduces GCD and autoattack delay by 13%) for 40s, +5 Kenki.
Gekko Single target third step combo. 100 potency, 150 from rear. If combo from Jinpu, increases to 320 potency, 370 potency from rear. Combo Bonus: Getsu Sen, +10 Kenki.
Kasha Single target third step combo. 100 potency, 150 from flank. If combo from Shifu, increases to 320 potency, 370 potency from flank. Combo Bonus: Ka Sen, +10 Kenki.
Yukikaze Single target second step combo. 100 potency, 280 potency if combo from Hakaze. Combo Bonus: Setsu Sen, +15 Kenki.
Enpi Single target ranged attack, 100 potency, 260 if combo (Yaten). +10 Kenki.

Global Cooldown Actions (Area of Effect)

Icon Name Description
Fuga AoE frontal cone, 90 potency. +5 Kenki.
Fuko Upgraded Fuga. AoE in a circle around you, 100 potency. +10 Kenki.
Mangetsu AoE in a circle around you, 100 potency, 110 if combo. Combo Bonus: Getsu Sen, Fugetsu buff. +10 Kenki.
Oka AoE in a circle around you, 100 potency, 110 if combo. Combo Bonus: Ka Sen, Fuka buff. +10 Kenki.

off-Global Cooldown Actions

Icon Name Description
Third Eye Reduces next damage taken by 10%. +10 Kenki if hit. 3s duration, 15s CD.
Meikyo Shisui Allows use of up to 3 Weaponskills without combo requirements. Grants Fugetsu on use of Gekko and Fuka on use of Kasha. Excludes laijutsu and Ogi Namikiri. 15s duration, 55s CD.
Meditate Channeled Kenki and Meditation stack generation for 15s. Initial cast on the GCD. +10 Kenki (Max 50) and +1 Meditation stack (Max 3) per tick. Cancelled upon movement or action. Can only be used in combat. 60s CD.
Hagakure Consumes all Sen, +10 Kenki for each consumed. 5s CD.
Ikishoten +50 Kenki. Can only be used in combat. Grants Ogi Namikiri Ready buff for 30s. 120s CD.
Hissatsu: Kaiten +50% potency to next Weaponskill. -20 Kenki. 15s CD.
Hissatsu: Gyoten Dash to target (20-yalm range), 100 potency. -10 Kenki. 10s CD
Hissatsu: Yaten 10-yalm backstep, 100 potency. Combos into Enpi. -10 Kenki. 10s CD.
Hissatsu: Shinten Single target attack, 270 potency. -25 Kenki. 15s CD.
Hissatsu: Kyuten AoE in a circle around you, 110 potency. -25 Kenki. 15s CD.
Hissatsu: Guren Line AOE, 500 potency with 50% damage falloff. -25 Kenki. 2m CD, shared with Senei.
Hissatsu: Senei Single target nuke, 800 potency. -25 Kenki. 2min CD, shared with Guren..
Shoha Single target attack, 580 potency. -3 Meditation stacks. 15s CD, shared with Shoha II.
Shoha II AoE in a circle around you, 200 potency. -3 Meditation stacks. 15s CD, shared with Shoha.

Iaijutsu

Icon Name Description
Iaijutsu 1.3s casted Weaponskill. Effect depends on the number of Sen held. +1 Meditation stack (Max 3).
Higanbana Single target 50s DoT. 200 potency initial hit, 30 potency ticks. Total 800.
Tenka Goken AoE frontal cone. 280 potency.
Midare Setsugekka Single target nuke. 660 potency.
Tsubame-gaeshi Ability, combos off laijutsu. Recasts previous laijutsu with higher potency. 60s CD. Can hold two charges.
Kaeshi: Higanbana Single target 60s DoT. 300 potency initial hit, 45 potency ticks. Total 1200. Does not stack with Higanbana.
Kaeshi: Goken AoE frontal cone. 420 potency.
Kaeshi: Setsugekka Single target nuke. 990 potency.
Ogi Namikiri AoE frontal cone. 800 potency with 75% damage falloff. +1 Meditation stack (Max 3).
Kaeshi: Namikiri AoE frontal cone. 1200 potency with 75% damage falloff.

Role Actions

Icon Name Description
Second Wind Instant self-heal for 500 potency. 120s cooldown.
Leg Sweep Stuns the target for 3s. 40s cooldown.
Bloodbath Heals self for 1/6 of all physical damage dealt for 20s. 120s cooldown.
Feint Reduces target’s physical damage dealt by 10% and magical damage dealt by 5% for 10s. 90s cooldown.
Arm’s Length Grants knockback immunity for 6s (note that some knockbacks cannot be mitigated with this) and inflicts 20% slow for 15s on attackers when struck during this time. 120s cooldown.
True North Nullifies all positional requirements for 10s. 45s cooldown, can hold 2 charges.

Traits

Icon Name Description
Kenki Mastery Unlocks the Kenki Gauge, increasing by 5 Kenki for using Gekko, Kasha, Oka, or Mangetsu and 10 Kenki for Yukikaze or being hit under Third Eye.
Kenki Mastery II Increases the Kenki Gauge by 5 Kenki for all weaponskills excluding Iaijutsu.
Enhanced Iaijutsu Reduces Iaijutsu cast time to 1.3 seconds.
Enhanced Fugetsu and Fuka Increases the effects of both Fugetsu and Fuka to 13%.
Enhanced Tsubame-gaeshi Allows Tsubame-gaeshi to accumulate charges to a maximum of two.
Fuga Mastery Upgrades Fuga to Fuko.
Enhanced Meikyo Shisui Allows Meikyo Shisui to accumulate charges to a maximum of two.
Enhanced Ikishoten Grants Ogi Namikiri Ready for 30s, allowing the use of Ogi Namikiri.
